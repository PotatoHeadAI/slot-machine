# 🎰 马铃薯头娱乐老虎机

一个纯娱乐性质的老虎机游戏，无任何赌博元素，仅供休闲娱乐。使用HTML5、CSS3和JavaScript构建。

## ✨ 游戏特点

### 🎮 纯娱乐性质
- **无赌博元素**：虚拟货币仅用于游戏内计分，无真实价值
- **教育意义**：展示概率和随机性的基本原理
- **休闲放松**：简单的点击游戏，缓解压力
- **成就系统**：收集虚拟奖章，增加游戏趣味性

### 🎨 游戏功能
- **老虎机转轮**：三个独立转轮，精美动画效果
- **多种符号**：8种不同符号，每种有不同的中奖倍数
- **虚拟投注**：四种投注额度选择（10、50、100、500虚拟币）
- **成就系统**：6个可解锁成就，奖励成就点数
- **游戏统计**：实时统计胜率、连胜、游戏时间等数据
- **音效支持**：旋转、中奖、虚拟币音效
- **响应式设计**：适配手机、平板、电脑各种屏幕

### 🔧 技术特性
- **纯前端实现**：无需后端服务器，零依赖
- **本地存储**：使用浏览器localStorage保存游戏进度
- **离线可用**：所有资源本地化，支持离线游戏
- **现代化UI**：CSS3动画，渐变背景，卡片设计
- **键盘支持**：空格键或回车键快速开始游戏

## 🎯 游戏规则

### 基本规则
1. **游戏目标**：三个转轮显示相同符号即可中奖
2. **虚拟货币**：初始1000虚拟币，仅用于游戏内计分
3. **中奖倍数**：不同符号有不同的中奖倍数（3-30倍）
4. **投注系统**：每局游戏消耗选择的投注额

### 符号与倍数
| 符号 | 名称 | 倍数 |
|------|------|------|
| 🍒 | 樱桃 | 3倍 |
| 🍋 | 柠檬 | 5倍 |
| 🍊 | 橙子 | 7倍 |
| ⭐ | 星星 | 10倍 |
| 🔔 | 铃铛 | 15倍 |
| 7️⃣ | 幸运7 | 20倍 |
| 🎰 | 老虎机 | 25倍 |
| 👑 | 皇冠 | 30倍 |

### 成就系统
完成特定任务可以解锁成就：
- **🥇 首胜**：赢得第一次游戏
- **💰 大赢家**：单次赢得500虚拟币
- **🔥 连胜大师**：达成5连胜
- **🎮 老玩家**：游玩50次游戏
- **🎯 头奖**：中得最高倍数（30倍）
- **🏆 收藏家**：解锁所有成就
